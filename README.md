# arcacondownloader
아카라이브콘 다운로더 입니다.

## 사용방법
1. arcacon.min.js 에 들어가 내용을 복사 합니다.
   ![Animation1](https://github.com/user-attachments/assets/fca4631d-7c31-445b-a54a-138f6563ce30)
2. 새로운 인터넷 창을 열어 북마크 추가를 합니다. (이름은 아무거나, URL은 앞에 `javascript:` 를 붙이고 복사했던 코드 붙여넣기)
   ![Animation](https://github.com/user-attachments/assets/1793204a-2b2e-4584-96cd-af8884ee60e1)
3. 다운로드 하고 싶은 아카라이브콘 페이지에 들어갑니다.
   ![Animation2](https://github.com/user-attachments/assets/33ab33d6-6970-45ed-8a33-085112c55207)
4. 아까 추가한 북마크를 눌러 다운로드 합니다. (다운로드가 안 되면, 압축파일 글씨를 클릭하면 다운로드 됩니다.)
   ![Animation3](https://github.com/user-attachments/assets/01d05290-4872-4345-ae5c-b93894b687db)
5. 다운로드 완료된 파일을 확인합니다.
   ![image](https://github.com/user-attachments/assets/71d2952d-46ee-4567-bb12-f10075c2aace)
